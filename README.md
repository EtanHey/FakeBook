## add a .env file including the following parameters:

> JWT_SECRET={your own string}

> PORT=4001

> MONGODB_URI={your mongoDB connection link}

## open terminal for server & client directories:

1. `npm i` for both.
2. `nodemon` for **server** & `npm start` for **client**.
